Ejecutar generador de build para proyecto Angular version 17
---> ng build
Pagina para servidor de prueba http:
--->https://www.npmjs.com/package/http-server
Comando a utilizar estar en super usuario:
---> npm install --global http-server
pegar ruta donde se encuentre el dist generado por el ng build
Y ejecutar el siguiente comando:
--->http-serve -o
para levantar proyecto de prueba.
